Important FAQ

What is the Document Object Model (DOM)?

HTML DOM (Document Object Model) is a hierarchical representation of HTML documents. It defines the structure and properties of elements on a webpage, enabling JavaScript to dynamically access, manipulate, and update content, enhancing interactivity and functionality.

How do you select an element from the DOM?

Using document.getElementById(id), document.getElementsByClassName(className), document.getElementsByTagName(tagName), document.querySelector(selector), document.querySelectorAll(selector).

What is event delegation in the context of the DOM, and why is it useful?

Event Delegation is basically a pattern to handle events efficiently. Instead of adding an event listener to each and every similar element, we can add an event listener to a parent element and call an event on a particular target using the .target property of the event object.It is useful for its improved performance, Dynamic Content Handling and Simpler Code Maintenance.

How do you manipulate an element's attributes and styles using the DOM?

In JavaScript, we can manipulate an element's attributes and styles using the DOM in various ways. Getting an Attribute:To get an attribute value of an element,we can use: getAttribute(attributeName),Setting an Attribute:To set or update an attribute, we can use:setAttribute(attributeName, value),Removing an Attribute:To remove an attribute, you can use: removeAttribute(attributeName), Setting Inline Styles:We can change an element's styles using the style object and setting the properties directly: For example: let element = document.getElementById("myElement");element.style.backgroundColor = "yellow"; element.style.fontSize = "20px"; Adding a Class: element.classList.add("newClass"); Removing a Class: element.classList.remove("oldClass").
